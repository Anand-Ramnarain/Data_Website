This is a slideshow, showcasing my wireframe. In this, the user can see the rough draw wireframes to how it became a blackout frame. This was important as it allowed me to have a visual blueprint of how I was going to start my website. It helped me to evaluate each element which I wanted to include whiles making sure it served a purpose on my website. Planning out the wireframes allow me to remove some doubt I had on what I wanted to include and achieve. In the wireframes, there is an orange circle on the frames, this is to represent a clickable object that will that the user to the top of the page.
