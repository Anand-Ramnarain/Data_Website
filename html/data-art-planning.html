Documentation For Game

Overview:

"Asteroid Defenders" is a space-themed game where players must defend a planet from a barrage of incoming asteroids. The game integrates real asteroid data from NASA's NEO (Near Earth Objects) API, offering a realistic representation of these celestial bodies.

Game Setup:

• The game is on a canvas.

• The canvas's center has a blue planet, symbolizing Earth.

• A protective cyan shield encircles the planet. This shield can take asteroid damage.

• Players control a green rectangular spaceship capable of shooting projectiles to demolish asteroids.

Game Objective:

The primary goal is to shield the planet from destructive asteroids. Players must:

• Eliminate asteroids by targeting them.

• Prevent asteroids from making contact with the planet's protective shield.

• Strive to defend the planet as long as possible. The game records this time and displays it on-screen.

Game Mechanics:

Asteroids:

• Asteroids, sourced from NASA's NEO API, hurtle from random trajectories towards the planet.

• The API provides size and other asteroid attributes, ensuring each game is different.

Spaceship Control:

• Navigate using arrow keys or WASD.

• Fire projectiles using the spacebar, F key, or by clicking the canvas.

Blasts:

• These blasts obliterate any asteroid they touch.

• Fire projectiles using the spacebar, F key, or by clicking the canvas.

Shield:

• A health bar for the shield is visible on the screen's top-left corner.

• An asteroid collision reduces the shield's health based on the asteroid's size.

• The game concludes when the shield's health depletes entirely.

Game States:

Start:

• The initial screen displays the game title with a "Play" button.

Running:

• Players control the spaceship, annihilate asteroids, and shield the planet.

Game Over:

• Triggered when the shield's health is fully depleted.

• Players view their survival time and can choose to replay the game.
