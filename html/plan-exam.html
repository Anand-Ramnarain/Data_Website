Documentation For Exam Data Art

Overview:

For my exam data art, I have decided to create data art that will track the ISS as it is moving in space and show it as it is moving. Here are my objectives to get this working:

Objectives:

1. Fetch real-time data about the International Space Station (ISS).

2. Visualize the ISS position on an SVG map.

3. Display information about the ISS location and provide city and country details.

4. Implement smooth transitions and updates every 5 seconds to reflect real-time changes.

The first thing I will have to do is:

Data Fetching:

By this, I need to use a fetchData function that fetches' API to get real-time ISS data so I can start creating my artwork

Visualization:

• once I get the data I will use D3.js for data binding, SVG manipulation, and visualization, this is so I can create the canvas and every component in it so it is functional.

• The SVG container will be updated to reflect the ISS position and a starry background will be created for visual appeal.

• Radial gradients are applied to the Earth and ISS circles for a realistic appearance.

Event Handling:

• Mouseover and mouseout event listeners are added to the Earth and ISS circles. This is so I can display information about the ISS location and city/country details displayed on mouseover so we can see whast city is it over.

Geocoding:

• I will create a display Location Info function that fetches additional location information using the API so I can show I can create a get Location Info function which will show the city and country information the ISS is over.

Info Box Display:

Then I will create a display info box to show the coordinate of the ISS when the user hovers over the ISS or the red dot represented in the art. When they hover over the planet, they can get information on which country it is above in space.

Continuous Updates:

To make sure the red dot represents the ISS is always moving getting real-time data of its position, I set a function to get its data every few minutes so that it will update.
